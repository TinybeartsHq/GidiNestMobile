# Custom font setup

The UI now references Neuzeit Grotesk families (Regular, Medium, SemiBold, Bold, ExtraBold).

To see the new typography in-app, drop the licensed font files in this folder using the
following filenames:

- `NeuzeitGro-Regular.ttf`
- `NeuzeitGro-Medium.ttf`
- `NeuzeitGro-SemiBold.ttf`
- `NeuzeitGro-Bold.ttf`
- `NeuzeitGro-ExtraBold.ttf`

After adding them, run `npx expo start -c` so Metro picks up the new assets.

> If you prefer another sans-serif, replace the files with fonts that share the same names.
